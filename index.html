Как это работает?

Что такое DAT?

Divergent Association Task — тест дивергентного мышления, разработанный учёными из Гарварда и Макгилла (Olson et al., 2021). Чем дальше по смыслу слова, которые ты придумал, тем выше балл.

Почему 7 слов?

Тест просит 10 слов, но оценивает только первые 7 валидных — остальные 3 служат буфером на случай невалидных. Это стандарт оригинального теста для сопоставимости результатов.

Как считается балл?

Из 7 слов составляются 21 пара. Для каждой пары вычисляется косинусное расстояние между векторами слов. Балл = среднее расстояние × 100.

Что такое расстояние?

Каждое слово — точка в 300-мерном пространстве (эмбеддинг). Близкие по смыслу слова находятся рядом, далёкие — далеко. Расстояние 0 = идентичны, 200 = максимально далеки.

Тепловая карта

Показывает расстояние между каждой парой из 7 слов. Тёмные ячейки — слова близки по смыслу, светлые — далеки.

Точность

Эмбеддинги взяты из модели navec (проект Natasha), обученной на русской литературе. Баллы приблизительные и не сопоставимы напрямую с английской версией DAT.
